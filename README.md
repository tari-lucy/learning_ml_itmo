# MeetingScribe — AI-секретарь совещаний

Веб-приложение для автоматической обработки аудио/видео записей совещаний, планёрок и созвонов. На выходе — готовый пакет документов: стенограмма с диаризацией, протокол совещания, список задач, краткое саммари.

## Что делает сервис

1. Пользователь загружает аудио/видео запись совещания
2. Система обрабатывает запись через цепочку ML-моделей
3. Пользователь получает готовые документы в личном кабинете

### Пайплайн обработки

```
Аудио/видео файл
    → Извлечение аудио (ffmpeg)
    → Транскрибация (faster-whisper)
    → Диаризация спикеров (pyannote-audio)
    → Формирование документов (LLM)
    → Пакет документов
```

### Документы на выходе

- Стенограмма — кто что сказал, с таймкодами
- Протокол совещания — повестка, решения, ответственные, сроки
- Список задач — кто / что / дедлайн
- Краткое саммари — ключевые пункты

## Стек технологий

Транскрибация - faster-whisper или аналог
Диаризация - pyannote-audio
Обработка аудио/видео - ffmpeg
Формирование документов - Deepseek / Mistral (Ollama)
Backend - FastAPI
База данных - PostgreSQL + SQLAlchemy
Очереди - RabbitMQ
Контейнеризация - Docker

## Функциональность

- Регистрация и авторизация пользователей
- Личный кабинет с балансом в кредитах
- Загрузка аудио/видео и запуск обработки
- Просмотр истории запросов и результатов
- REST API для внешних интеграций
- Валидация входных данных
- Роль администратора (модерация пополнений, просмотр транзакций)

## Объектная модель

| Класс | Описание |
|---|---|
| `MLModel` | Базовый класс ML-моделей (→ WhisperModel, DiarizationModel, SummaryModel) |
| `User` | Пользователь системы (→ Admin) |
| `Transaction` | Базовый класс транзакций (→ DebitTransaction, CreditTransaction) |
| `Task` | Задача на обработку аудио |
| `Result` | Результат обработки (пакет документов) |
| `MeetingScribeService` | Сервисный класс, координирующий работу системы |
